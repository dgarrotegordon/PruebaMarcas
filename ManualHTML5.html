1.- Estructura

Crear un documento con HTML5 es bastante sencillo si conoces su estructura.
En primer lugar, debemos colocar la etiqueta <!DOCTYPE html>, que no tiene etiqueta de cierre. Después colocamos la etiqueta <html> y la cerramos(</html>). Y dentro de esta etiqueta es donde escribiremos nuestro documento completo.
Nuestro documento se dividirá en tres secciones:

HEAD
Será la cabecera de nuestro documento. Se declara usando <head> y podemos declarar el título(<title>), la codificación(<meta charset="UTF-8">) y otros elementos como enlaces, estilos, etc.

BODY
Como su propio nombre indica, esto será el cuerpo o contenido general del documento. Es lo que el navegador nos mostrará al abrirlo.

FOOT
En este apartado introduciremos los estilos CSS que queramos para dar formato a la página que estemos creando con el documento.

2.- HEAD

En la cabecera podemos encontrar:

Title(<title>)
Define un título para el documento
Meta(<meta>)
Define un metadato de un documento, en este caso le indicaremos la codificación del documento al navegador
Lang(<lang>)
Definimos un idioma para el documento
Style(<style>)
definimos un estilo para la información del documento

3.- BODY

En el cuerpo encontraremos:

Elementos de bloque:
Contienen elementos de línea o de bloque, siempre comienzan en una nueva línea.
Elementos de línea:
Contienen solo datos o elementos de línea.

3.1.- Manejo de Texto

A continuación se nombrarán varios elementos de bloque o línea:

  • 3.1.1.- Encabezados(<h1> a <h6>)

    • Cómo su propio nombre indica, establecen encabezados o títulos. Es un elemento de línea y se clasifican con números del 1 al 6, siendo h1 el más grande y h6 el más pequeño.
    Veamos como quedaría cada uno de ellos:

    h1

    h2

    h3

    h4

    h5
    h6
  • 3.1.2.- Párrafo

    • Esta etiqueta agrupa todas las líneas de su interior. Dentro de esta etiqueta, los saltos de línea deben ser explícitos. Esto lo conseguimos con la etiqueta <br> para un salto de línea en blanco o con la etiqueta <hr> si queremos visualizar una línea horizontal, que separe un párrafo de otro elemento. También tenemos la etiqueta <wbr> para crear saltos de línea en un palabra por si el navegador reduce el formato del texto.
    Tenemos también otro tipo de etiquetas para el manejo de texto que se deben poner dentro de un párrafo.

    • Negrita usando <b>.
    • Bastardilla/Cursiva/Itálica usando <i>.
    • Subrayado usando <u>.
    • Tachado usando las etiquetas <s> o <strike>.
    • Texto mayor usando <big>.
    • Texto menor usando <small>.

    Finalmente, podemos representar un texto de forma que el navegador lo muestre tal como aparece en el documento. De esa forma, respetaría las tabulaciones, saltos de línea o espacios en blanco. Esto se haría utilizando la etiqueta <pre>.

3.2.- Manejo de Listas

  • 3.2.1.- Lista de elementos no ordenada

    • En el caso de las listas no ordenadas, tenemos que usar dos etiquetas. La primera etiqueta que debemos usar es la que define el tipo de lista <ul>. Dentro de esta etiqueta, debemos aplicar la etiqueta <li> para cada elemento. A parte de esto, con el atributo "type" podemos cambiar el tipo de símbolo utilizado delante de cada elemento(p.e. <ul type="disc">). Así se vería una lista de este tipo con diferentes tipos de símbolos.

    • Atributo type "square" coloca un cuadrado como símbolo.
    • 2º Elemento

    • Atributo type "disc" coloca un punto negro.
    • 2º Elemento

    • Atributo type "circle" coloca un punto con fondo blanco.
    • 2º Elemento


  • 3.2.2.- Lista de elementos ordenada

    • las listas ordenadas, al igual que las no ordenadas, se componen de dos etiquetas. La primera es la que indica el tipo de lista <ol>, y dentro de esta definiremos cada elemento de la lista con la etiqueta <li>(como en las listas no ordenadas). Este tipo de lista puede tener varios atributos:

    Atributo type:
    Al igual que en las listas no ordenadas, nos permite elegir el tipo de símbolo o carácter que precede a los elementos de nuestra lista.

    p.e. <ol type="I">
    1. Elemento 1
    2. Elemento 2
    3. Elemento 3

    Atributo start:
    Este atributo nos permite especificar en que número o carácter comenzará a contar nuestra lista.

    p.e. <ol type="I" start="3">
    1. Elemento 1
    2. Elemento 2
    3. Elemento 3

    Atributo reversed:
    Finalmente tenemos el atributo reversed. Con este atributo podemos hacer que nuestra lista se numere de forma inversa.

    p.e. <ol reversed start="3">
    1. Elemento 1
    2. Elemento 2
    3. Elemento 3


  • 3.2.3.- Lista de definiciones

    • En el caso de las listas de definiciones tenemos tres etiquetas:

    • Etiqueta que indica el tipo de lista: <dl>
    • Etiqueta que marca el término que se va a definir: <dt>
    • Etiqueta que acota la definición: <dd>

    Cabe añadir que este tipo de etiquetas no es muy común en la red.

3.3.- Enlaces

También llamados hipervínculos, sirven para establecer una referencia a otro elemento del mismo u otro documento de manera que nos lleve directamente a él.
Estos enlaces se acotan con la etiqueta <a>, y deben llevar el atributo "href" para indicar su destino.
La manera de escribir correctamente un enlace sería asi: <a href="url_de_destino">nombre_del_enlace</a>
Podemos poner cómo ejemplo este enlace al buscador de Google.

3.4.- Insertar Imágenes

Para insertar imágenes en nuestro documento, debemos usar la etiqueta <img> acompañada del atributo "src" y la dirección de la imagen. Con los atributos width y height indicamos el tamaño en el que se visualizará la imagen y con alt le damos un nombre para casos en los que el navegador no permita visualizar la imagen.
Esta sería una etiqueta genérica: <img src="url_imagen" width="ancho_px" height="alto_px" alt="nombre_imagen">
Como ejemplo pondré la etiqueta y debajo la imagen que se visualizará.

<img src="http://www.w3.org/html/logo/downloads/HTML5_Logo_512.png" width="150px" height="150px" alt="HTML 5">


3.5.- Crear Tablas

Para crear una tabla tenemos que considerar primero cuantas filas y columnas necesitamos. A parte si queremos alguna fila de encabezado o algun elemento adicional. Para crear una tabla debemos usar la etiqueta <table>. esta etiqueta cuenta con etiquetas adicionales como <tr> para colocar filas y dentro de estas <td> para crear celdas debtro de estas filas.
Los atributos más importantes son:

  • Atributos de la tabla

    • El más utilizado es el atributo border que sirve para definir si la tabla lleva borde o no. En anteriores versiones de HTML se le daban valores, en HTML 5 simplemente se le asigna el valor 1 ó 0 dependiendo de si queremos borde o no, el tamaño del borde se debe hacer con estilos en CSS. Para los bordes internos usaremos atributos rules. Tambien los datos de las celdas se pueden ajustar con el atributo align hacia la derecha, centro o izquierda.

  • Atributos de las celdas:

    • rowspan, que sirve para denominar el número de filas que ocupa esa celda, y colspan, que se utiliza para denominar el número de columnas que ocupa.

  • Atributos de las filas:

    • En este apartado entrarían los atributos thead, que define una fila como cabecera de la tabla, tbody que definiría una o varias filas como el cuerpo de la tabla y tfoot que sería para definir una línea como pie de tabla.

A continuación pondremos el ejemplo de una tabla en HTML 5, de manera que veamos un poco mejor la manera de crearla.

3.6.- Crear Mapa de Imágenes

Los mapas de imágenes se utilizan para hacer interactivo el documento al visualizarlo en el ordenador. Usaremos la etiqueta <area> añadida a la etiqueta <img> para especificar una región de la imagen y el vínculo que asociaremos a esa región. A parte tenemos el atributo shape que nos permite especificar una región(rect, circle,etc.) con coordenadas especificando la posición en la pantalla. En caso de un rectángulo daríamos las cordenadas de la esquina superior izquierda(x,y) y de la esquina inferior derecha(x,y), en el caso de un cículo indicaríamos el centro(x,y) y una tercera cifra para el radio

Aquí se muestra un ejemplo de un mapa de imágenes:

El primer paso sería insertar la imagen, y dentro colocar el atributo usemap de esta manera: <img usemap="#nombre_mapa" src="https...>
Luego añadiríamos la etiqueta <map> y el atributo shape para colocar un par de enlaces en las coordenadas de la imagen que queramos. Así que escribiremos lo siguiente:
<map name="mapa">
<area shape="tipo_area" coords="cifras coord_separadas_por_comas" href="url_vínculo">
Esto podemos repetirlo tantas veces como vínculos queramos poner en la imagen
No nos olvidemos de cerrar la etiqueta </map>
